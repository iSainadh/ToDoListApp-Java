# To-Do List CLI Application (Java)

A simple command-line application to manage daily tasks efficiently.

## 🚀 Features
- Add, update, delete, and view tasks
- Mark tasks as completed
- Menu-driven interface

## 🛠️ Technologies
- Java
- OOP Principles
- ArrayList

## 📦 How to Run
```bash
javac ToDoListApp.java
java ToDoListApp
